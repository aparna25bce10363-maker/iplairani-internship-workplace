
# IPLAIRANI Internship Workplace

This repository serves as the central workspace for all internship assignments at IPLAIRANI.

All interns are required to complete assigned tasks and submit their work in a structured, trackable manner using Git.

---

## Objective

The purpose of this repository is to:

- Provide real-world project experience  
- Ensure structured and consistent submissions  
- Enable efficient progress tracking  
- Introduce interns to professional Git workflows  

---

## Repository Structure


tasks/
(Contains all assigned problem statements)

submissions/
(Each intern must create their own folder here)

---

## Getting Started

### Step 1: Clone the Repository

      git clone <your-repo-url>
---

### Step 2: Navigate into the Repository

    cd iplairani-internship-workplace

---

### Step 3: Go to Submissions Folder

    cd submissions

---

### Step 4: Create Your Folder

Create a folder using your name:

    mkdir your_name

---

### Step 5: Add Your Work

    * Place all your code inside your folder
    * Follow the assigned task from `/tasks`

---

### Step 6: Commit Your Changes

    git add .
    git commit -m "Added initial project submission"

---

### Step 7: Push Your Code

    git push origin main

---

## Task Instructions

* All tasks are available in the `/tasks` folder
* Read the task carefully before starting
* Follow all requirements mentioned

---

## Submission Guidelines

Each submission must include:

* Source code
* A `README.md` inside your folder explaining:

  * What you built
  * How to run it
  * Any assumptions

---

## Rules

* Do not modify other interns’ folders
* Do not delete any existing files
* Work only inside your own folder
* Commit regularly with meaningful messages

---

## Expectations

* Clean and readable code
* Proper folder structure
* Working implementation
* Clear documentation
