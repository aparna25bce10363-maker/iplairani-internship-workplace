## How to Submit Your Work

We understand that not everyone is familiar with Git.  
So you can choose **any one of the following methods**:

---

### Using Git (Recommended)

1. Fork this repository  
2. Clone your fork:

       git clone <your-fork-url>


3. Navigate to submissions:

       cd submissions

4. Create your folder using your Gmail ID:

       mkdir yourname@gmail.com

5. Add your code inside this folder
6. Commit and push:

       git add .
       git commit -m "Added my submission"
       git push origin main

7. Create a Pull Request (PR)

---

### Option 2: Using Google Drive

If you are not comfortable with Git:

1. Create a folder with your name
2. Add your project files inside it
3. Upload it to Google Drive
4. Share the link with us
